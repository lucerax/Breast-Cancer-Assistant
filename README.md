# Breast Cancer Assistant
Diagnostic assistant for histopathological diagnosis of breast cancer. Intended as a teaching assistant for medical registrars. 

Can classify H&E stained tissue slides for four histological distinct types of benign breast tumors: adenosis (A), fibroadenoma (F), phyllodes tumor (PT), and tubular adenona (TA);  and four malignant tumors (breast cancer): carcinoma (DC), lobular carcinoma (LC), mucinous carcinoma (MC) and papillary carcinoma (PC).
